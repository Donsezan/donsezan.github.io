What This Page Does

This tool optimizes prompts or code by removing unnecessary symbols to reduce character count while keeping meaning intact.

Buttons:
• LLM Optimize: Removes redundant whitespace.
• Code Optimize: Removes spacing around code symbols.
• Clear: Clears input.
• Copy: Copies output.
• History panel stores session inputs.
